# PHP XML Processing - Student Management

## Objective
To demonstrate XML file creation, insertion of data, and reading XML data using PHP SimpleXML.

## Features
- Creates XML file (students.xml)
- Adds student details dynamically
- Displays student data in table format

## Files
- `index.php` - Main PHP file
- `students.xml` - XML data storage

## How It Works
1. User enters student details (Name, Age, Course)
2. Data is added to XML file
3. XML file is read and displayed in table

## Run Instructions
1. Place folder inside `htdocs`
2. Start Apache
3. Open:

   http://localhost/9-xml-processing/index.php

## Output
- Form to add student
- XML file gets updated
- Table shows all students

## Example
Input:
Name: Yokesh  
Age: 18
Course: CSE  

Output:
Displayed in table and stored in XML
